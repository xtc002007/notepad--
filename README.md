# notepad--

一个基于 Tauri 的现代记事本应用。

## 特性

- 跨平台支持（Windows、macOS、Linux）
- 现代化界面
- 轻量快速

## 本地运行

**前置要求：** Node.js

1. 安装依赖：
   ```bash
   npm install
   ```

2. 运行应用：
   ```bash
   npm run dev
   ```
